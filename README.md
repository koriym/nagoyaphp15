## 雛形

### 1. インストール

```bash
$ git clone git@github.com:ttskch/nagoyaphp15.git
$ cd nagoyaphp15
$ composer install
```

### 2. 実装

[Nagoyaphp\Dokaku15\Dokaku15::run()](/src/Dokaku15.php#L9) を適切に実装する。

### 3. テスト

```bash
$ composer test
```
